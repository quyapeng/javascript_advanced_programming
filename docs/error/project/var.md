相比于其他语言，JavaScript中的变量可谓独树一帜。正如ECMA-262所 规定的，JavaScript变量是散类型的，而且变量不过就是特定时间点一 个特定值的名称而已。由于没有规则定义变量必须包含什么数据类型， 变量的值和数据类型在脚本生命期内可以改变。这样的变量很有意思， 很强大，当然也有不少问题。本章会剖析错综复杂的变量。
